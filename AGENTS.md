# Agent Rules

- Do not start the dev server in this repository. The user starts and manages it.
- Before pushing changes that will trigger deployment, run the same root checks
  as CI: `pnpm run lint` and `pnpm run typecheck`. If a check cannot be run or
  fails, do not deploy/push as "done"; fix it or report the blocker.
- Story generation: do not center stories around sweets, desserts, candy, or
  treat-style rewards. Prefer everyday activities, family outings, practical
  skills, nature, vehicles, reading, building, and other non-sweets topics.
  Incidental food should be ordinary/balanced and not framed as the main
  reward.
